# 资源贡献指南

欢迎为 Origin of Miracles 项目贡献资源内容！

## 贡献类型

- [人格配置](./personas.md)：编写学生的 AI 对话行为
- [模型制作](./models.md)：创建 GeckoLib 3D 模型和动画

## 快速开始

1. 阅读 [创作者守则](./code-of-conduct.md)
2. Fork [Anima-Assets 仓库](https://github.com/Origin-of-Miracles/Anima-Assets)
3. 根据类型选择对应指南
4. 提交 Pull Request

## 重要提示

- 所有内容必须遵守 **[CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/)** 许可证
- 严格遵守 [创作者守则](./code-of-conduct.md)，禁止 R18/政治/血腥内容
- 提交前使用验证工具检查格式
