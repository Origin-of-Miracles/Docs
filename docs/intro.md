# Welcome to Origin of Miracles

**Where all miracles begin. | 一切奇迹的始发点。**

欢迎来到 Origin of Miracles 文档站！我们是一个致力于将 **《蔚蓝档案》(Blue Archive)** 的沉浸式体验带入 Minecraft 世界的开发者组织。

## 🎯 项目简介

Origin of Miracles 是一个**非官方粉丝项目**，通过革命性的技术栈（Webview + React + Forge Mod）将原作的 UI 美学与养成系统完美复刻到方块世界中。

我们的目标只有一个：**让爱丽丝（和其他的所有学生）在方块的世界里真正"活"过来。**

> "老师，今后也请多多关照！" — *天童爱丽丝*

## 🛠️ 核心组件

我们采用 **Forge Mod + Webview** 的分离式架构：

| Component | Description |
| :--- | :--- |
| **Miracle Bridge** | 核心模组，注入 Chromium 容器，提供 JS ↔ Java 双向通信 |
| **Shittim OS** | 运行在游戏内的 React SPA，包含 MomoTalk、抽卡等界面 |
| **Server Pack** | 基于 Forge 的服务端配置，包含地图、枪械数据等内容 |
