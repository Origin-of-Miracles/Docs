## 🛠️ 生态系统

我们采用 **Forge Mod + Webview** 的分离式架构，构建了一个轻量且强大的交互生态：

| Component | Codename | Stack | Description |
| :--- | :--- | :--- | :--- |
| **Miracle Bridge** | `Client` | Java (Forge 1.20.1) | 核心模组。负责注入 Chromium 容器，提供原生 JS ↔ MC 双向通信接口 |
| **Shittim OS** | `Webview` | React + TypeScript | 沉浸式前端。运行在游戏内 Webview 中的 React SPA 应用 |
| **Server Pack** | `World` | TacZ + Scripts | 服务端整合。包含基沃托斯地图构建、枪械数据平衡及任务脚本 |
